## 🎓 어떤 파일을 선택할까?

### 📘 **사용 시나리오별 추천**

**상황 1: 팀 전체에 배포하고 싶다**
```
→ 웹협업운영기준_통합백서.docx 다운로드
  - Word 형식이므로 수정 가능
  - 인쇄/이메일 공유 용이
  - 모든 스타일이 적용된 최종 버전
```

**상황 2: 웹사이트/협업 도구에 올리고 싶다**
```
→ 웹협업운영기준_통합백서.pdf 다운로드
  - PDF는 모든 브라우저에서 열림
  - 수정 불가능 (고정된 형식)
  - 구글 드라이브/노션에 임베드 가능
```

**상황 3: 내용을 커스터마이징하고 싶다**
```
→ 협업기준_강화버전.md 다운로드
  - 마크다운 포맷 (텍스트 에디터로 편집)
  - 섹션을 추가/삭제 가능
  - 회사/팀에 맞게 수정 가능
```

**상황 4: 미리 확인하고 싶다**
```
→ preview-01.jpg + preview_table-*.jpg 확인
  - 최종 레이아웃 확인
  - 실제 렌더링 결과 보기
```

## ❓ FAQ

**Q: Word와 PDF 중 뭘 다운로드해야 하나요?**  
A: 대부분의 경우 **Word 문서**를 추천합니다. 수정 가능하고, 필요시 PDF로 변환할 수 있습니다.

**Q: 마크다운 버전으로 뭘 할 수 있나요?**  
A: 회사 맞춤형으로 수정하거나, GitHub에서 버전 관리하거나, 자동화 도구로 변환할 수 있습니다.

**Q: 이 문서를 어떻게 사용해야 하나요?**  
A: 팀 전체에 공유하고, "이 기준에 따라 협업합시다"라고 합의하세요. 각 역할(기획/디자인/개발)이 자신의 체크리스트를 사용하면 됩니다.

**Q: 수정하고 싶으면?**  
A: Word를 받아서 직접 편집하거나, Markdown을 다운로드해서 텍스트 에디터로 수정하세요.
