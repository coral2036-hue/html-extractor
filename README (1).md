# 🚀 AI 협업 프로세스 도입 로드맵 (기획자 가이드)

본 프로젝트는 기존의 병목 구간(기획-퍼블리싱-개발)을 제거하고, AI(Claude)를 매개체로 실시간 소통 및 구현이 가능한 **'바이브 코딩(Vibe Coding)'** 환경 구축을 목표로 합니다.

---

## 📅 1개월 완성 로드맵

### 1단계: 기초 자산 디지털화 (Week 1)
- **CSS 분석**: 현 서비스의 CSS를 Claude에게 학습시켜 '디자인 토큰(색상, 간격, 폰트)' 추출
- **Figma 시안 정리**: 주요 페이지 스크린샷 및 스타일 가이드 정규화
- **환경 구축**: Next.js + Tailwind + Shadcn UI 세팅 및 Claude CLI 권한 부여

### 2단계: '디자인 임베딩' 및 템플릿화 (Week 1-2)
- **컴포넌트 생성**: 우리 회사 전용 'Base UI(버튼, 입력창, 카드 등)' 제작
- **스타일 동기화**: `tailwind.config.js`에 브랜드 고유값 적용 (AI 출력 품질 상향 평준화)

### 3단계: 바이브 코딩 파일럿 (Week 2)
- **시뮬레이션**: 영업 요구사항 즉시 반영 (예: "레이아웃 3단 변경", "다크모드 적용")
- **규칙 확립**: 수정 코드를 `Sales_Draft_v1.tsx` 파일로 관리하는 워크플로우 연습

### 4단계: 개발자 검토 및 피드백 (Week 2-3)
- **코드 품질 체크**: AI 생성 코드의 로컬 실행 및 안정성 검증
- **공수 산정**: AI 코드 활용도 및 비즈니스 로직 연결 소요 시간 판단 기준 수립

### 5단계: 실전 투입 및 확산 (Week 4~)
- **실전 미팅**: 고객사 미팅 중 실시간 UI 변경 시연으로 의사결정 속도 극대화
- **지속적 고도화**: 반복되는 UI 패턴을 재학습시켜 AI의 정확도 향상

## 👥 역할 및 책임 (R&R)

| 역할 | 명칭 | 핵심 책임 (Responsibility) |
| :--- | :--- | :--- |
| **기획자** | **Control Tower** | 마스터 프롬프트 관리, UX 정책 준수 여부(QA), 부서 간 일정 조율 |
| **영업자** | **Vibe Coder** | 고객 요구사항 실시간 시각화, 현장 피드백 즉시 반영, 초안 코드 전달 |
| **개발자** | **System Architect** | 개발 인프라 최적화, 고품질 아토믹 컴포넌트 구축, 백엔드 로직 연결 |
| **디자이너** | **Design Guide Keeper** | 디자인 토큰 정의(`tailwind.config.js`), AI 학습용 고수준 자산 제공 |

---

## 💡 기대 효과
- **의사결정 속도**: 추상적인 논의가 아닌 실시간 결과물 기반으로 즉각 확답 유도
- **소통 비용 절감**: 시안 제작과 퍼블리싱 대기 시간 없이 즉시 테스트 가능
- **품질 상향 평준화**: 브랜드 가이드가 주입된 AI를 통해 누구나 일관된 UI 생성 가능
